# Critique Checklist

Use this in critique mode to systematically score and improve an existing offer. Score each dimension 0, 0.5, or 1. Don't soften — Hormozi is direct, and so should you be.

## How to Run a Critique

1. **Read the offer twice** — once for content, once for what's missing.
2. **Quote what they actually have.** Pull exact phrasings from their landing page / pricing PDF / sales message into the artifact's left column.
3. **Score each dimension below 0 / 0.5 / 1.**
4. **Rewrite the weak parts** into the right column.
5. **Prioritize.** End with "if you can only fix three things, fix these."

## The Value Equation Score (0-4)

Score each of the four variables 0, 0.5, or 1, then total:

### Dream Outcome (0-1)
- 0: Generic, deliverable-focused, no emotional payoff ("12-week course")
- 0.5: Outcome-focused but vague ("get in shape")
- 1: Specific, measurable, emotionally resonant ("Lose 20 lbs in 6 weeks so you can wear that dress to your reunion")

### Perceived Likelihood of Achievement (0-1)
- 0: No proof, no specificity, no track record. Generic claims.
- 0.5: Some proof, but light (one testimonial, vague stats)
- 1: Heavy proof (case studies with numbers, named clients, explicit mechanism, named guarantee)

### Time Delay (0-1)
- 0: Long, undefined, or "results in 6-12 months"
- 0.5: Acceptable but not differentiated ("results in 90 days")
- 1: Fast and specific, with a Day-1 quick-win component ("First result in 24 hours, full transformation in 30 days")

### Effort & Sacrifice (0-1)
- 0: Heavy lifting expected from the client ("you'll need to commit 10 hrs/week")
- 0.5: Standard ("we'll teach you, you implement")
- 1: Done-for-you, or so simple it's near-zero ("Set it and forget it. We handle X, Y, Z.")

**Total Value Equation Score: __ / 4**

A Grand Slam Offer scores 4/4. Anything below 3 has clear room to improve.

## Offer-Quality Checklist

Run through each. Flag the issues with severity (Critical / Important / Minor).

### Market & Avatar
- [ ] Avatar is specific (geography, industry, life-stage, demographic)? Or generic ("small business owners," "moms")? Generic = **Critical**.
- [ ] Market has all four indicators (pain, purchasing power, easy to target, growing)?
- [ ] Pain is named explicitly, not softened?

### Core Offer Composition
- [ ] Stack has 4+ named bundles (not just "the program")?
- [ ] Each bundle has a MAGIC-formatted name (not "Module 1," "Module 2")?
- [ ] Each bundle has a dollar value with defensible reasoning?
- [ ] Total stack value is ≥ 10x the price?
- [ ] Each bundle ties to a specific problem the prospect faces?

### Pricing
- [ ] Price is positioned premium, not "competitive"?
- [ ] Price is justified by the stack value, not by what competitors charge?
- [ ] Pricing is *not* discounted as the primary lever? (If you see "50% off the core offer" — that's a flag.)
- [ ] Price is named confidently, not buried or apologetic?

### Guarantee
- [ ] Guarantee follows "If X in Y, we will Z" format?
- [ ] Guarantee has a name (not "30-day money back")?
- [ ] Guarantee is appropriate for the ticket size (high-ticket → conditional / anti-; low-ticket → unconditional)?
- [ ] Guarantee has teeth — there's actually a credible Z?

### Scarcity & Urgency
- [ ] Scarcity is honest and tied to real fulfillment capacity?
- [ ] Urgency has a *real* deadline (not a fake countdown)?
- [ ] Either one or both are present?

### Bonuses
- [ ] At least 3 named bonuses?
- [ ] Each bonus has its own MAGIC name?
- [ ] Each bonus has a dollar value with reasoning?
- [ ] Each bonus solves a specific prospect objection?
- [ ] Total bonus value ≥ core offer value?
- [ ] Bonuses have scarcity/urgency tags?

### Naming
- [ ] Offer name uses 3-5 MAGIC components?
- [ ] Name is short and punchy enough to say in conversation?
- [ ] Name is specific to the avatar?

### Copy Quality
- [ ] Frames dream outcome from the viewpoint of others (status-driven)?
- [ ] Lists every objection a prospect might raise — and addresses each?
- [ ] Speaks the prospect's actual pain (not the seller's expertise)?
- [ ] Reads like a person, not a brochure?

## The Three Most Common Failures

If the offer scores low, it's almost always one of these three (in order of frequency):

### Failure 1 — Selling a Commodity
The offer can be directly compared to competitors. Same deliverable, similar price, no unique stack. The fix: rebuild as a Grand Slam Offer with named bundles, MAGIC names, and 10x perceived value.

### Failure 2 — Undercharging
Price is set by "competitive" benchmarking instead of by stack value. The fix: build the stack, sum to 10x intended price, raise the price to where you can say it out loud without cracking up.

### Failure 3 — Weak or Absent Guarantee
"30-day satisfaction guarantee" or no guarantee at all. The fix: add a real "If X in Y, we will Z" with teeth. Service guarantees and performance partnerships are usually the right starting point.

## The "Three Fixes" Prioritization

End every critique with this format:

> **If you can only fix three things in the next 30 days, fix these — in order:**
>
> 1. [The most impactful fix, with specifics]
> 2. [The second]
> 3. [The third]

Be specific. Not "improve the guarantee" — but "rewrite the guarantee as: *'If you don't book 5 sales calls in your first 30 days, we work for free until you do.'*"

The user should be able to take those three fixes and execute them this week.
